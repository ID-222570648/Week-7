Reflection: Custom events passes from child component
to parent the parent component. In this example, the base or parent component is the app,
and the child component is component 1. Component 1 emits custom event, which is called random.
The base component 'app' receives return value of the method call by the event random, and the
value is transfered and displayed at the base component.
Custom event is a good way to transfer data to the parent componet.
